# Scorciatoie da tastiera VS Code per programmare alla velocità della luce

Visual Studio Code è l'editor più popolare al mondo, e conoscerne le scorciatoie può raddoppiare la tua velocità di scrittura. Ecco le più utili!

### 1. Editing di base

`Ctrl + C` / `Ctrl + X` - copia o taglia l'intera riga (senza selezione)

`Alt + ↑ / ↓` - sposta la riga corrente su o giù

`Shift + Alt + ↑ / ↓` - duplica la riga corrente su o giù

`Ctrl + Enter` - inserisce una riga sotto

`Ctrl + Shift + Enter` - inserisce una riga sopra

`Ctrl + /` - commenta / de-commenta la riga

`Alt + Click` - aggiungi cursori multipli (multicursore)

`Ctrl + D` - seleziona la prossima occorrenza della parola corrente

### 2. Navigazione

`Ctrl + P` - vai al file (ricerca rapida per nome)

`Ctrl + G` - vai alla riga numero...

`Ctrl + Shift + O` - vai al simbolo (funzione, classe, variabili) all'interno del file

`F12` - vai alla definizione

`Alt + F12` - sbircia la definizione (senza aprire il file)

`Ctrl + Home` / `End` - vai all'inizio / fine del file

### 3. Gestione finestre e vista

`Ctrl + B` - mostra / nascondi la barra laterale

`Ctrl + \` - dividi l'editor (split view)

`Ctrl + 1, 2, 3` - passa al gruppo di editor 1, 2 o 3

`Ctrl + Shift + E` - mostra l'esplora risorse (files)

`Ctrl + Shift + F` - cerca in tutti i file (search)

`Ctrl + ` (backtick) - mostra / nascondi il terminale integrato

### 4. Comandi e refactoring

`Ctrl + Shift + P` - Command Palette (esegui qualsiasi comando VS Code)

`F2` - rinomina simbolo (variabile, funzione) in tutto il progetto

`Ctrl + .` - Quick Fix (suggerimenti per errori o import mancanti)

`Shift + Alt + F` - formatta il documento

### Bonus: Zen Mode

`Ctrl + K Z` - attiva la Zen Mode per concentrarti solo sul codice

---

Impara 2-3 di queste scorciatoie a settimana e vedrai la tua produttività volare!
